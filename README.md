# MSCS 532 Final Project: HPC Data Locality Optimization

## Overview
This project demonstrates the impact of data locality optimization in High-Performance Computing (HPC).

## Objective
Compare performance between:
- Linked list (poor memory locality)
- Array (good memory locality)

## Key Concept
Data locality improves cache utilization and reduces execution time.
